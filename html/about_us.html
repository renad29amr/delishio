Project Objective

The objective of this project is to create an interactive, user-friendly website where users can browse, share, and manage food recipes from various cuisines.The platform allows users to create an account, log in securely, and explore a wide range of recipes complete with images and step-by-step instructions.An admin can manage website content to ensure that only relevant and high-quality recipes are displayed.

Project Functionalities

  • User Registration:
    • Registration form with client-side validation using JavaScript and regular expressions.
    • Ensures valid email, password strength, and proper name formatting.
  • Login & Logout:
    • Secure login system for registered users.
    • Logout functionality to ensure session security.
  • Home Page:
    • Attractive homepage with navigation menu and image slideshow highlighting featured recipes.
    • Easy access to all other sections of the website.
  • Recipe Browsing:
    • Users can view categorized recipes (e.g., Breakfast, Lunch, Dinner, Desserts, Drinks).
    • Each recipe page includes ingredients, preparation steps, and an image.
  • Admin Page:
    • Admin can add, edit, or delete recipes.

Technologies Used

  • HTML – Structure of web pages
  • CSS – Styling and layout
  • JavaScript – Interactivity and client-side validation
